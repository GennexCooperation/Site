Heptaengine - Basics

Installation instruction:
Before you can actually start creating your game, you firstly need to install Heptaengine.
After you have installed Heptaengine.zip rightclick it and press on extract.
You'll see a couple .py files which we only focus on "engine.py" and "game.py"
engine.py is the file where you will develop your game while you use game.py to run the game
.py files can't usually be read on PCs by default. For this you need a Python-Interpreter which can be installed doing following:

Windows:
-for windows you can just go to the Microsoft Store, search for Python and download it. Alternative you can go to their website and download it there, which is a bit more complicated.

Linux:
(debian based): sudo apt install python3
(fedora): sudo dnf install python3
(arch): sudo pacman -S python3

Mac:
-just visit their website or download via homebrew

Before you start actually working on any game, it is important to know a couple basic commands.

>> create [gamename]
with this simple command you create main files of your games
(you can't name your game "placeholder" since placeholder is a premade constant game with important files to create new games)

>> edit [gamename]
with this command you choose what game you want to start editing an existing file you created with "create [gamename]".
(again the game with the name placeholder cannot be edited)

>> remove [gamename]
with this command you can remove games fully that were created.
(placeholder is not removable)

>> location
with this command you can check what game you are currently editing
